HTML таблиці використовуються для представлення даних у форматі рядків і стовпців, що робить їх зручними для відображення структурованої інформації, такої як таблиці з даними, статистикою чи звітами.

Основні елементи таблиць:

  • <table>: оголошує саму таблицю.
  • <tr> (table row): визначає рядок таблиці.
  • <th> (table header): визначає заголовок стовпця (зазвичай текст відображається жирним шрифтом та по центру).
  • <td> (table data): визначає клітинку з даними в рядку таблиці.

Приклад: 

                        <table border="1" cellpadding="10" cellspacing="0">
                        <thead>
                            <tr>
                                <th>Назва книги</th>
                                <th>Автор</th>
                                <th>Жанр</th>
                                <th>Ціна</th>
                            </tr>
                        </thead>
                        <tbody>
                            <tr>
                                <td>Хроніки Нарнії</td>
                                <td>Клайв Льюїс</td>
                                <td>Фентезі</td>
                                <td>500 грн</td>
                            </tr>
                        </tbody>
                    </table>

HTML дозволяє створювати списки для структурування контенту. Існує два основних типи списків: нумеровані (упорядковані) і ненумеровані (неупорядковані).

Основні елементи списків:

  • <ul>: ненумерований список (unordered list).
  • <ol>: нумерований список (ordered list).
  • <li>: пункт списку (list item).

Приклад ненумерованого списку: 

                            <ul>
                                <li>Пункт 1</li>
                                <li>Пункт 2</li>
                            </ul>

Приклад нумерованого списку: 

                            <ol>
                                <li>Перше дію</li>
                                <li>Друге дію</li>
                            </ol>

HTML-код зображень використовується для вставлення графіки в веб-сторінку. Зображення можуть бути статичними або анімованими, і вони зазвичай використовуються для ілюстрації контенту, прикраси або надання додаткової інформації.

Основні елементи зображень:

  • <img>: використовується для вставлення зображення. Основні атрибути:
    • src: вказує URL-адресу зображення.
    • alt: альтернативний текст, який відображається, якщо зображення не може бути завантажено, або використовується для доступності.

Приклад: 

                            <img src="path/to/image.jpg" alt="Опис зображення" width="300" height="200">

Десктопна версія

Десктопна версія оптимізована для екранів шириною понад 1200px.

Для десктопів використовується широка структура макета із центральним контентом, шрифтами стандартного розміру та великою кількістю простору між елементами.

Пояснення підходу

Для адаптивної верстки використані брейкпоінти 768px та 1024px. Для кожного розміру екрану створено окремі стилі, що забезпечують оптимальний UX. Головні аспекти:

  • Великий розмір шрифтів та просторі блоки для десктопів.
  • Оптимізація відступів для планшетів.
  • Спрощення структури для мобільних пристроїв.

Логічні оператори

Логічні оператори (and, not, or) у медіа-запитах дозволяють комбінувати кілька умов для точнішої настройки стилів для різних пристроїв. Це дає можливість створювати гнучкіші та більш оптимізовані дизайни для кожного типу екрана.

            function makeArray(firstArray, secondArray, maxLength) {
                // Створюємо новий масив шляхом об'єднання firstArray і secondArray
                const combinedArray = firstArray.concat(secondArray);
            
                // Якщо довжина об'єднаного масиву перевищує maxLength, повертаємо його копію з обмеженням по довжині
                if (combinedArray.length > maxLength) {
                    return combinedArray.slice(0, maxLength);
                }
            
                // Якщо довжина об'єднаного масиву в межах maxLength, повертаємо весь масив
                return combinedArray;
            }
            
            // Приклади використання
            console.log(makeArray([1, 2], [3, 4, 5], 4)); // [1, 2, 3, 4]
            console.log(makeArray(["a", "b"], ["c", "d", "e"], 3)); // ["a", "b", "c"]
            console.log(makeArray([10, 20], [30, 40], 5)); // [10, 20, 30, 40]

function makeArrayC(A, B) {
    let C = [];
    for (let i = 0; i < A.length; i++) {
        if (A[i] === B[i]) {
            C.push(1);
        } else {
            C.push(1 / (A[i] - B[i]));
        }
    }
    return C;
}

function swapMaxAndFirst(C) {
    let maxIndex = C.indexOf(Math.max(...C));
    let temp = C[0];
    C[0] = C[maxIndex];
    C[maxIndex] = temp;
    return C;
}

            // Функція для сортування масиву методом бульбашки
function bubbleSort(arr) {
    let n = arr.length;
    // Проходимо по всіх елементах масиву
    for (let i = 0; i < n - 1; i++) {
        // Проходимо через усі елементи масиву
        for (let j = 0; j < n - 1 - i; j++) {
            // Якщо поточний елемент більший за наступний, міняємо їх місцями
            if (arr[j] > arr[j + 1]) {
                let temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
    return arr;
}

// Масив для сортування
let inputArray = [7, 2, 9, 3, 5, 1, 4, 8, 6];

// Виведення вхідного масиву
console.log("Вхідний масив:", inputArray);

// Викликаємо функцію сортування
let sortedArray = bubbleSort([...inputArray]); // використовуємо spread operator для копії масиву

// Виведення відсортованого масиву
console.log("Вихідний масив (відсортований):", sortedArray);

        function Storage(initialItems) {
    this.items = initialItems;

    // Метод для отримання всіх товарів
    this.getItems = function () {
        return this.items;
    };

    // Метод для додавання нового товару
    this.addItem = function (item) {
        this.items.push(item);
    };

    // Метод для видалення товару
    this.removeItem = function (item) {
        const itemIndex = this.items.indexOf(item);
        if (itemIndex !== -1) {
            this.items.splice(itemIndex, 1); // Видаляє товар з масиву
        }
    };
}

// Приклад використання
const arr = ["apple", "banana", "mango"];
const storage = new Storage(arr);

storage.addItem("orange");
console.log(storage.getItems()); // ["apple", "banana", "mango", "orange"]

storage.removeItem("banana");
console.log(storage.getItems()); // ["apple", "mango", "orange"]

            function checkBrackets(str) {
                const stack = [];
                const brackets = {
                    ')': '(',
                    '}': '{',
                    ']': '[',
                };
            
                for (const char of str) {
                    // Якщо символ - відкрита дужка, додаємо в стек
                    if (['(', '{', '['].includes(char)) {
                        stack.push(char);
                    }
                    // Якщо символ - закрита дужка
                    else if ([')', '}', ']'].includes(char)) {
                        // Перевіряємо, чи стек не порожній і верхній елемент у ньому відповідає закритій дужці
                        if (stack.pop() !== brackets[char]) {
                            return false;
                        }
                    }
                }
            
                // Якщо стек порожній, дужки збалансовані
                return stack.length === 0;
            }
            
            // Приклади використання
            console.log(checkBrackets("function someFn() { return [1, 2, 3]; }")); // true
            console.log(checkBrackets("function someFn() { return [1, 2, 3; }"));  // false
            console.log(checkBrackets("(a + b) * {c / [d - e]}"));                 // true
            console.log(checkBrackets("(a + b) * {c / [d - e]}]"));                // false
